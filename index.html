NBA Doom

Make a Team of 3 NBA superstars including yourself

A web application that allows users to create players and add them to a team of 3 including NBA players based off of NBA 2k ratings. Similar to the NBA2k video game series, a user would be able to open a "player pack," which includes 3 NBA players. The odds of any getting any given player is random and a user can choose to add the newly acquired player to their team.

Technologies used include: JavaScript, HTML, CSS, Ruby, JQuery, Ruby on Rails, PostgreSQL, Bootstrap.
